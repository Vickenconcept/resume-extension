# Quick Fix: Create Icon Files

The extension needs icon files to load. You have 3 options:

## Option 1: Use the HTML Generator (Easiest)

1. **Open `generate-icons.html` in your browser**
   - Double-click the file or right-click → Open with → Browser
   
2. **Click "Download All Icons" button**

3. **Move the downloaded files** to this `icons` folder
   - icon16.png
   - icon48.png
   - icon128.png

## Option 2: Create Simple Icons Manually

Use any image editor (Paint, Photoshop, GIMP, etc.) to create:

- **icon16.png** - 16x16 pixels, orange background (#f97316), white "R" letter
- **icon48.png** - 48x48 pixels, orange background (#f97316), white "R" letter  
- **icon128.png** - 128x128 pixels, orange background (#f97316), white "R" letter

## Option 3: Use Online Icon Generator

1. Go to any online icon generator (e.g., favicon.io, realfavicongenerator.net)
2. Create a simple icon with:
   - Background: Orange (#f97316)
   - Text/Logo: White "R"
3. Download and resize to the required sizes
4. Save as icon16.png, icon48.png, icon128.png

---

**After creating the icons, try loading the extension again!**

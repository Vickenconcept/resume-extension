# Extension Icons

Place your extension icons here:
- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

For now, you can use any 16x16, 48x48, and 128x128 PNG images, or create simple placeholder icons.

Suggested design: Orange and white color scheme matching the extension theme.
